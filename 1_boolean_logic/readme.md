### 그리고 아래 3가지 파일을 작성하고 테스트하고 cmp 파일을 만들어야 한다.
- \[칩 이름\].hdl : 하드웨어 시뮬레이터에서 쓰는 칩 스크립트
- \[칩 이름\].tst : 테스트 스크립트
- \[칩 이름\].cmp : 테스트 스크립트의 output을 기록한 파일
